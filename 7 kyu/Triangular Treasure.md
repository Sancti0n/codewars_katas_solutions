## JavaScript
```js
function triangular( n ) {
  if (n<=0) return 0
  return n*(n+1)/2
}
```

## Java
```java
public class Triangular {
  public static long triangular(long n) {
    if (n<=0) return 0;
    return n*(n+1)/2;
  }
}
```
